# 🧮 Interactive Calculator

A simple, interactive calculator built with HTML, CSS, and JavaScript. Features full keyboard support and responsive design.

## ✨ Features
- ➕ Addition, ➖ Subtraction, ✖️ Multiplication, ➗ Division, 🧮 Percentage
- ⌨️ Full keyboard support
- 📱 Responsive design
- 🎨 Dark theme

## 🚀 Quick Start
Open `index.html` in your browser

## ⌨️ Keyboard Shortcuts
- `0-9` → Numbers
- `+ - * / %` → Operations
- `Enter` → Calculate
- `Backspace` → Delete
- `Escape` → Clear

## 📁 Files
- `index.html` - HTML structure
- `style.css` - Styling
- `calculator.js` - Calculator logic

## 💻 Technologies
HTML5, CSS3, JavaScript
